# backend

This repository contains a backend that allows you to schedule experiments by defining a json that contains parameters for the experiment, chiefly among which: which exact LLM to use, with which settings, what prompt and what schema. The experiments are saved to a postgres database. 

The [orchestration repo](https://git.science.uu.nl/auto-kg-lit/orchestration) contains instructions on how to deploy the backend.
